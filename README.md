# Kinly Web

Marketing website for the Kinly / Health Tracker app.

## Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- GSAP / Framer Motion / Lenis (animation + smooth scroll)

## Quick start

```
cd web
npm install
npm run dev
```

## Layout

- `web/` — the Next.js app
- `.kiro/`, `.claude/` — local tooling for hyperframes / GSAP / website-generation skills

## Origin

Split out of the main `Health Tracker` repo on 2026-04-27 to keep the
mobile app, the marketing site, and the promo videos in separate
repositories. The mobile app stays at `Health Tracker/apps/mobile/`.
